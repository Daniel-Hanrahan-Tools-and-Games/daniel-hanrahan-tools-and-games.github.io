How I Became a successful Tool Builder/Developer

  1. Find a need or find a problem that people need solving.
  2. Build the hardware possibly needed for that tool.
  3. Figure out the logic for the software possibly needed for that tool.
  4. Write the software possibly needed for that tool.
  5. Find all problems with hardware and software and fix them until it is perfected or near perfected.
  6. Release/Publish the tool.
  7. Tell people about it on Reddit on the correct subreddit for the tool or equivalents.
  8. Listen to feedback and if there is anything wrong with the tool fix it and continue to make fixes and updates.
